# Mutual Fund AI Frontend

Modern Next.js frontend for AI-powered mutual fund recommendations.

## Quick Start

```bash
npm install
npm run dev
```

App runs on http://localhost:3000

## Features

- 5 professional pages with modern UI
- Real-time backend integration
- Interactive charts and visualizations
- Responsive design for all devices
- AI-powered recommendations

## Pages

- **Home** - Landing page with statistics
- **Dashboard** - Market overview with charts
- **Recommendations** - AI-powered fund selection
- **Analysis** - Advanced statistical insights
- **Fund Explorer** - Search and compare funds

## Tech Stack

- Next.js 16 with TypeScript
- Tailwind CSS
- Recharts for data visualization
- Lucide React for icons
